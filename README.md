# Solar System App

A beautiful animated solar system displaying the planets orbiting around the sun.

## How to Run

Simply open `index.html` in your web browser to view the animation.

## Features

- Animated planets orbiting the sun
- Realistic colors for each planet
- Smooth animation using HTML5 Canvas

## Files

- `index.html`: The main HTML file
- `style.css`: Styles for the page
- `script.js`: JavaScript for the animation
